indent 不是合成词，而是一个源自拉丁语的单词，意思是“凹陷”或“缩进”。它在英语中作为一个独立的单词使用，表示文本的首行或其他部分的缩进。 关于 indent： 词源：indent 来源于拉丁语“indentare”，意思是“切口”或“凹陷”。它后来演变为英语中的“indent”，主要用于描述文本排版中的首行缩进。 在 CSS 中： text-indent：一个 CSS 属性，用于设置文本块中第一行的缩进量。这个属性专门用于控制块级元素的首行缩进。 合成词（compound words）通常由两个或更多的词组成，如“toothbrush”（牙刷）或“sunflower”（向日葵）。indent 并不属于这种情况，它是一个单一的词，不是由其他词组合而成的合成词。
